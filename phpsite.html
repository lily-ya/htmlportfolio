Interactive Olympics Website

An interactive website providing searching and browsing information about Olympic Games

Main Role: Full-stack Developer
Skills: PHP, MariaDB, Python
Client: Database Management Class
Team Size: 4
Date: December 2017
Github: Code

The project was created in Fall 2017 for Database Management Class. Our team aims to create a PHP website that interacts with MariaDB using SQL statements to provide information of Olympic Games. Our website includes 5 pages and the database involves 6 tables. Data were collected from Wikipedia and olympic.org using a customized python script. In addition, we use a bootstrap framework to make the website responsive to various devices.


Entity Relationship Diagram. We modified the diagram instantly as we adding more tables and refining our information. The diagram provides a clear relationships between entities, and helps improve coding efficiency.



The home page includes the information of the upcoming games and google map showing the location of the hosting country.



The browse page includes Olympic games events since 1896.



An info page is linked by clicking on event thumbnail.



The search page allows 4 filtering methods, and the searching results are linked to the responding info page.
