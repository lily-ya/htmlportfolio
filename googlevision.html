User Engagements Prediction

Application of sentiment analysis in predicting user engagements with features extracted from the images using Google Cloud Vision API

Main Role: Data Analyst
Skills: Python (selenium, nltk, pandas, numpy)
Class: User Generated Content Analysis
Team Size: 6
Date: November 2017
Github: Code

Images (and videos) contribute much more to big data than text (or numbers). Is a picture worth a thousand words? In this project, we run logistic regression models to predict user engagement and reactions based on the image tags extracted using Google Cloud Vision API. User engagement is measured by the number of likes and comments. User reactions is measured by the average sentiment of its comments. We downloaded 750 images along with the number of likes, the number of comments, author's description, and latest 20 comments using Selenium with Python. The conclusion of the project is that the choose of image affects user's attention.


We choose images from National Geographic on Instagram because there's so much variation in the composition of their photography.



To quantify the user engagement of the images, we combined normalized number of likes and comments with weights to create an engagement score. Then, we categorized the engagement score with 1 if the score is above the mean, and 0 otherwise.



Photo that has the highest average sentiment scores on the comments. We used VADER sentiment analysis tools to obtain the sentiment of the comments.



Image combined with author's description can effectively predict user's engagement. Also, content of the image is strongly associated with polarity of user's comments.
