Visual Question Answerability Classification

Predicting if a question is answerable for a given image using Hand-Crafted features and Neural Networks

Main Role: Data Scientist
Skills: Python
Class: Intro to Machine Learning
Team Size: 1
Date: March 2018
Dataset: VizWiz
Code: Azure Notebook

To predict whether a visual question is answerable, I created a multi-model classification system. The Neural Network is trained using four categories of features: 1) Image-based features extracted using the Azure Vision API. 2) Question-based features extracted using the Azure Text Analytics API. 3) Detecting if there is readable words on the image using Azure OCR API. 4) TF-IDF features are transformed from the questions using Python's scikit libray. The model was trained and validated using the original data split. The validation accuracy reaches 70%.


The model was trained using 19977 examples with 3705 features in total, 3698 of them are TF-IDF features, and the rest 7 features are hand picked from the data returned by Azure API. The first two column features are based on questions. numKeyPhrases counts the number of key phrases based on the question text. Sentiment is the sentiment score for the question text. aldultScore and racyScore indicate if the content of the image is adult content. isBw indicates if the image is black and white. has_ocr_text indicates if there are readable text on the image.
